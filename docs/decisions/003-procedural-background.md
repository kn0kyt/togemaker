# ADR-003: トゲトゲ背景は Canvas で手続き生成する

## ステータス

Accepted（2026-07-13）

## コンテキスト

『ガールズバンドクライ』のトゲトゲ演出（井芹仁菜の怒りのオーラ）風の背景が本ツールの核。
アニメのフレーム（©東映アニメーション）をそのまま素材にするのは著作権上 NG。

## 決定

背景は静止画素材を使わず、Canvas でパラメータ駆動の手続き生成にする（`src/toge-background.ts`）。
黒ベース + 赤いモヤ + 赤黒シャード（外周ほど高密度・中央は空ける）+ 色収差 + グレインの
レイヤー構成で「風」を自前描画する。

## 理由

- 著作物を一切含まないため権利的に安全（非公式・非営利のファンメイドとして明記する前提）
- シード付き乱数により「🎲 再生成」で毎回違う絵が出せる
- 「怒りゲージ」（トゲの量・長さ・赤み・中央への迫り方）のパラメータ化と相性が良い。
  原作でも感情の強さに応じてトゲの量が変わるため、原作準拠のマッピングになる
- 代替案（自作の静止画 1 枚）は手軽だが、固定絵で再生成・強度調整ができない

## 影響

- 見た目の原作再現度はパラメータ調整に依存する（参考画像との比較チューニングを継続）
- 背景の生成コストがあるため、被写体操作中は背景キャッシュを使い再生成しない設計にした
