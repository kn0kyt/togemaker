# Project Guide

## プロジェクト概要

togemaker（トゲメーカー / ローカルフォルダは暫定 `toge`）: 写真をアップロードすると背景が自動で切り抜かれ、『ガールズバンドクライ』の井芹仁菜の「トゲトゲ」演出風（黒背景 + 赤黒の鋭利な破片）の背景に合成できる、ブラウザ完結・サーバーレスの画像メーカー。InspirationCat（キュピーン猫画像メーカー）のオマージュ。

### 主要機能
- 画像アップロード → ブラウザ内 AI（ONNX Runtime Web）で背景自動切り抜き
- Canvas で手続き生成したトゲトゲ背景への合成（強度スライダー・ランダム再生成）
- 被写体の移動・拡縮・回転、PNG 保存

### 技術スタック
- Vite + TypeScript（フレームワークなし）+ Canvas API
- 背景除去: `@imgly/background-removal`（ISNet / ONNX Runtime Web）※ライセンス確認中
- ホスティング: Cloudflare Pages（サーバー・DB なし、運用費 0 円）

### 制約
- 画像は外部送信しない（全処理ブラウザ内）
- 東映アニメーションの素材は使用しない（自前描画の「風」表現・非公式ファンメイド明記）

詳細は `docs/spec.md` を参照。
